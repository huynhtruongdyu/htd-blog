AiBiet (meaning "AI knows?" in Vietnamese) is a modular, AI-powered CLI assistant and developer utility tool. Built with C# and Spectre.Console, it offers a rich and interactive terminal experience.

Key features:
- **Multi-model AI support** — Query and chat with Ollama, Gemini, and OpenAI providers
- **Security scanner** — AI-powered vulnerability scanning
- **Translation tool** — AI-powered text translation between languages
- **Commit generator** — Generate conventional commit messages with AI
- **Developer utilities** — GUID generation, formatting, and more
- **Health check** — Diagnose system, providers, and connectivity
